# AcousticSpace Deepfake Detection

## Project Overview
AcousticSpace Deepfake Detection is a machine learning project that aims to identify AI-generated or manipulated audio recordings. The system analyzes acoustic features from speech signals and classifies them as either genuine or deepfake using deep learning techniques.

## Objectives
- Detect deepfake audio recordings.
- Extract meaningful audio features.
- Train and evaluate a deep learning model.
- Provide an easy-to-use prediction interface.

## Tech Stack
- Python
- TensorFlow / PyTorch
- Librosa
- NumPy
- Pandas
- Scikit-learn
- Streamlit
- Git & GitHub
