# Model Development

## Project
AcousticSpace: Deepfake Detection via Room Impulse Response (RIR)

## Objective
Develop a deep learning model to detect AI-generated speech by analyzing acoustic characteristics using a pretrained Audio Spectrogram Transformer (AST).

## Responsibilities
- Dataset loading
- Audio preprocessing
- Baseline CNN development
- AST fine-tuning
- Model evaluation
- Confidence score generation
- Model export for FastAPI integration

## Folder Structure
- dataset/ : Dataset loader and metadata
- models/ : Model architectures and saved weights
- scripts/ : Helper scripts
- utils/ : Utility functions
- checkpoints/ : Intermediate training checkpoints
- outputs/ : Evaluation results and plots
- notebooks/ : Experimental notebooks
